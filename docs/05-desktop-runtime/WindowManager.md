# Window Manager

## 职责

Window Manager 负责桌宠窗口的系统级能力。

## MVP 能力

- Windows 透明窗口
- macOS 透明窗口
- 无边框
- 基础置顶
- 可拖动
- 显示与隐藏
- Windows 托盘恢复
- macOS 菜单栏恢复
- 按平台保存窗口位置

## 平台约束

| 能力 | Windows 实现重点 | macOS 实现重点 |
| --- | --- | --- |
| 窗口置顶 | 避免被普通窗口覆盖 | 处理 Space 和全屏应用下的表现差异 |
| 透明窗口 | 确认点击区域和透明背景 | 确认阴影、圆角和透明层级 |
| 恢复入口 | 系统托盘 | 菜单栏状态项 |
| 位置记忆 | 按显示器坐标保存 | 按屏幕可见区域校正 |

任何平台能力失败时，必须降级为“普通可见窗口 + 可恢复入口”，不能导致应用无法启动。

## 后续能力

- 鼠标穿透切换
- 多显示器位置记忆
- 全屏应用检测
- 开机启动

## 错误处理

系统能力失败时必须返回明确错误码，前端展示用户可理解提示。

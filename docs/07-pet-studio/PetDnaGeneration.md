# Pet DNA Generation

## 输入

- 主图照片，必填
- 参考图照片，选填，最多 4 张
- 用户填写名称
- 用户选择或 AI 识别的物种
- 可选描述

## 多图策略

- 主图用于建立宠物的默认外观和预览。
- 参考图用于补充主图无法稳定识别的局部特征。
- 多张图片必须来自同一只宠物。
- AI 需要输出识别依据摘要，帮助用户理解为什么生成这些字段。
- 当图片之间存在明显冲突时，生成结果应降低置信度，并提示用户删除异常参考图后重试。

## 输出

- Pet DNA JSON
- 识别置信度
- 图片一致性提醒
- 关键识别依据
- 可编辑字段

## 校验

AI 输出必须校验：

- species 是否在支持范围内
- name 是否为空
- personality 是否在枚举内
- favoriteFoods 是否过长

## 降级

生成失败时，进入手动创建流程。

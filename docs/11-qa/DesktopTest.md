# Desktop Test

## 测试项

- Windows 透明窗口
- macOS 透明窗口
- 置顶
- 拖动
- 隐藏与显示
- Windows 托盘退出
- macOS 菜单栏退出
- 重启恢复位置

## 场景

- 普通桌面
- 多显示器
- 全屏应用
- 高 DPI

## 验收

- Windows 桌宠不能阻塞用户正常使用其他应用。
- macOS 桌宠不能阻塞用户正常使用其他应用。
- 两个平台资源加载失败时都必须回退到 Momo Pet。

# Milestone

## M0：知识库完成

完成条件：

- 文档目录结构稳定。
- MVP 范围明确。
- API、数据库、AI、桌面端核心方案成文。
- 后续编码任务可直接从 SprintPlan 拆出。

## M1：首次创建闭环

完成条件：

- 用户能创建宠物。
- 用户能上传照片。
- 系统能生成或手动填写 Pet DNA。
- 用户能看到宠物详情。

## M2：桌宠可运行

完成条件：

- Tauri 桌宠窗口透明置顶。
- 宠物可拖动、隐藏、退出。
- 至少支持三种动作状态。

## M3：养成闭环

完成条件：

- 喂食、铲屎、抚摸影响状态。
- 状态能持久化。
- 失败场景有明确错误提示。

## M4：AI 陪伴闭环

完成条件：

- 用户能聊天。
- 宠物回复符合 Pet DNA。
- 聊天历史可保存并生成摘要。

## M5：Beta 包

完成条件：

- Windows 安装包可分发。
- 数据可恢复。
- 有基础日志和错误上报。
- 10 名测试用户完成 7 天体验。

# Glossary

## Pet DNA

宠物数字身份。包含外观、性格、喜好、讨厌项、口头禅、声音风格等长期稳定信息。

## Desktop Runtime

桌面运行时。负责透明窗口、置顶、托盘、拖动、动作播放和桌面交互。

## Pet Life

宠物生命系统。负责饱食度、心情、清洁度、能量、亲密度、等级和成长。

## Pet Studio

宠物工坊。负责上传照片、识别宠物、生成 Pet DNA 和生成桌宠资源。

## Pet Story

宠物故事系统。根据互动、状态和记忆自动生成日记、时间线和纪念内容。

## Care Action

照顾行为，包括喂食、铲屎、抚摸、洗澡、玩耍等。

## AI Generation Task

AI 生成任务。用于异步处理 Pet DNA、头像、动作、故事和摘要生成。

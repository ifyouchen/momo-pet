# 00 Vision

## 产品名称

Project Momo，中文暂称「Momo AI 宠物」。

## 产品愿景

每一只宠物，都值得拥有数字生命。

Project Momo 不是普通桌宠，也不是一组循环动画。它是一款 AI Native Desktop Pet：用户导入真实宠物照片后，系统生成专属数字宠物。这个宠物可以在 Windows 和 macOS 桌面陪伴用户，拥有身份、性格、记忆、情绪、成长和日常行为。

## 一句话定位

上传宠物照片，AI 自动生成属于你的 Windows/macOS 桌面宠物，它会陪伴、成长、聊天、提醒和记录你们的共同生活。

## 目标用户

- 猫、狗、鸟、兔子等宠物主人。
- 希望纪念已离世宠物的用户。
- 喜欢桌宠、电子宠物和陪伴型 AI 的用户。
- 长时间在电脑前工作、学习、写代码的用户。
- 愿意为情感陪伴、个性化内容和长期记忆付费的用户。

## 核心差异化

传统桌宠的价值主要是动画展示，Project Momo 的价值是数字陪伴：

- 真实宠物照片生成专属宠物，而不是固定角色。
- Pet DNA 让每只宠物拥有可持续演化的数字身份。
- 长期记忆让宠物越用越懂主人。
- 桌面深度交互让宠物参与用户的真实电脑生活。
- Pet Story 自动记录宠物和主人的共同故事。

## 产品北极星指标

用户在 7 天内主动唤起或互动桌宠的次数。

该指标比注册数、生成数更能说明产品是否提供了陪伴价值。

## 商业模式方向

免费版提供一只宠物、基础动作、基础聊天和本地存档。

Pro 版提供：

- 多只宠物
- 云同步
- AI 长期记忆
- 高级动作
- 更多食物与装扮
- Pet Story 绘本
- 高质量资源重新生成
- 多设备同步

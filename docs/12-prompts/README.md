# Prompt Library

## 目标

Prompt Library 是给 AI 编码和 AI 生成功能使用的版本化资产。

MVP 阶段先建设少量核心 Prompt，不追求 200 个文件。

## 当前 Prompt

- [petdna-generate-v1.md](petdna-generate-v1.md)
- [chat-reply-v1.md](chat-reply-v1.md)
- [memory-summary-v1.md](memory-summary-v1.md)
- [code-review-v1.md](code-review-v1.md)

## 规范

每个 Prompt 必须包含：

- 目的
- 输入
- 输出
- 约束
- 示例

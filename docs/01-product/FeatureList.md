# 03 Feature List

## P0 功能

| 功能 | 业务目的 | 验收标准 |
| --- | --- | --- |
| 上传宠物照片 | 获取真实宠物输入 | 支持 JPG、PNG、WebP，失败有错误提示 |
| 创建宠物档案 | 建立宠物身份 | 名称、类型、生日、性格可保存 |
| 生成 Pet DNA | 为 AI 行为提供统一输入 | 可生成并展示关键字段 |
| 桌宠显示 | 形成核心陪伴体验 | Windows/macOS 均支持透明、置顶、可拖动 |
| 喂食 | 建立养成反馈 | 饱食度、心情、经验变化 |
| 铲屎 | 增加照顾感 | 清洁度恢复，心情变化 |
| 抚摸 | 提供即时互动 | 亲密度增长，播放反馈动作 |
| 基础聊天 | 建立 AI 陪伴感 | 可发送消息并返回宠物口吻回复 |
| 状态保存 | 支持长期养成 | 重启后状态不丢失 |

## P1 功能

- Windows 系统托盘
- macOS 菜单栏入口
- 开机启动
- 宠物主动提醒
- 工作状态检测
- 多只宠物切换
- 宠物动作资源管理
- 本地 SQLite 离线运行
- 后端云同步

## P2 功能

- Pet Story 自动日记
- 宠物相册
- 装扮系统
- 商城
- 插件 SDK
- 创作者市场
- 多设备同步
- 语音对话

用文字线框定义 MVP UI，后续可据此生成 Figma 或直接实现 React 组件。

## 聊天面板

```text
┌──────────────────────────────┐
│ 和奶糖聊天                    │
│                              │
│ 奶糖：今天也要陪我玩哦。       │
│                              │
│ 你：今天工作好累              │
│ 奶糖：辛苦啦，我陪你休息一下。 │
│                              │
│ [ 输入消息...              ]  │
│                      [ 发送 ] │
└──────────────────────────────┘
```

## 状态反馈

喂食成功：

```text
+ 饱食度 15
+ 心情 5
+ 经验 10
```

聊天失败：

```text
我刚刚走神了，再说一次好吗？
[ 重试 ]
```

## 组件尺寸约束

- 互动面板宽度建议 280 到 360 px。
- 聊天面板宽度建议 320 到 420 px。
- 桌宠本体不得因为动画帧变化导致窗口跳动。
- 按钮文字必须完整显示。

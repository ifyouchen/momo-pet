# Pet Interaction UI

## 目标

定义宠物交互相关 UI 元素，包括气泡、状态浮层、食物面板、清理提示、抚摸反馈、聊天入口和离线提示。

## UI 原则

- UI 不抢宠物本体的注意力。
- 所有浮层尺寸小、停留短、可自动消失。
- 文案短，动作明确。
- 操作控件要适合重复使用。

## 气泡 Bubble

### 用途

- 宠物主动短句。
- 操作反馈。
- 错误轻提示。

### 结构

```text
┌──────────────────┐
│ 主人，摸摸我。    │
└─────────▲────────┘
          │
        宠物
```

### 规则

- 单行优先。
- 最多 2 行。
- 不超过 20 个中文字符。
- 默认 4 到 6 秒后消失。
- 鼠标移入时可延长 2 秒。

### 类型

| 类型 | 用途 |
| --- | --- |
| Normal | 普通短句 |
| Success | 操作成功 |
| Warning | 饥饿、困倦 |
| Error | 可恢复错误 |

## 状态变化浮层

### 用途

展示操作带来的数值变化。

示例：

```text
+ 饱食 18
+ 心情 8
+ 经验 10
```

### 位置

显示在宠物右上方或互动面板操作区附近。

### 动效

- 轻微上浮。
- 1.5 到 2 秒淡出。
- 减少动画模式下只淡入淡出。

## 食物选择面板

### 布局

```text
┌────────────────────────┐
│ 选择食物                │
│                        │
│ [小鱼干]  饱食 +18      │
│ [鸡胸肉]  饱食 +25      │
│ [牛奶]    饱食 +12      │
│ [零食]    心情 +10      │
└────────────────────────┘
```

### 字段

- 食物图标。
- 食物名称。
- 主要效果。
- 不可用原因。

### 状态

- Normal：可点击。
- Loading：当前食物提交中。
- Disabled：已吃饱或离线不可用。
- Error：提交失败。

## 清理事件提示

### 展示

清理事件在桌宠附近显示一个小型提示，不使用过度恶心的视觉。

```text
┌────────────┐
│ 需要清理    │
│ [清理]      │
└────────────┘
```

### 规则

- 不遮挡宠物主体。
- 用户清理后立即消失。
- 无清理事件时不显示。

## 抚摸反馈

### MVP

点击抚摸按钮后：

- 按钮进入短 loading。
- 宠物播放 Happy。
- 状态浮层显示 intimacy、mood 增量。
- 可显示爱心或轻量粒子，但不能铺满屏幕。

### 减少动画模式

只显示状态浮层，不播放粒子。

## 聊天入口

### 入口形式

- 互动面板中的聊天按钮。
- 宠物气泡中的快捷回复入口，Beta 后支持。

### 聊天按钮状态

| 状态 | 表现 |
| --- | --- |
| Normal | 可打开聊天 |
| Unread | 有宠物主动消息 |
| Loading | 正在发送或等待回复 |
| Offline | 禁用并显示离线提示 |

## 聊天面板

### 结构

```text
┌──────────────────────────────┐
│ 和奶糖聊天              [×]  │
├──────────────────────────────┤
│ 奶糖：今天也要陪我玩哦。       │
│ 你：今天工作好累              │
│ 奶糖：辛苦啦，我陪你休息一下。 │
├──────────────────────────────┤
│ [ 输入消息...              ]  │
│                      [发送]  │
└──────────────────────────────┘
```

### 消息样式

- 用户消息右侧。
- 宠物消息左侧。
- 宠物消息可带小头像。
- 失败消息显示重试。

## 互动面板按钮

### 按钮

- 喂食
- 铲屎
- 抚摸
- 聊天
- 设置

### 规则

- 使用图标 + 文本。
- 文本不能被截断。
- Loading 时保留按钮宽度。
- 禁用状态必须说明原因。

## 离线提示

### 全局离线条

在操作页面顶部显示：

```text
当前离线，已显示最近状态。
```

### 局部离线提示

聊天、上传、AI 生成入口禁用，并显示：

```text
联网后才能使用。
```

## 宠物状态标签

### 标签

- 开心
- 饿了
- 困了
- 想玩
- 很干净
- 需要清理

### 规则

标签来自 Life Engine，不由前端自行判断复杂业务。

## Loading 设计

### AI 分析

使用阶段文案：

- 正在识别颜色
- 正在分析性格
- 正在生成 Pet DNA

### 操作请求

按钮内 loading，不使用全屏 loading。

## Error 设计

错误优先使用轻提示。

只有以下情况使用面板错误：

- 创建失败。
- 上传失败。
- AI 任务失败。
- 缓存损坏。

## 尺寸约束

- Bubble 最大宽度 220 px。
- 食物面板宽度 260 到 320 px。
- 聊天面板宽度 320 到 420 px。
- 状态浮层不超过 3 行。

## 验收标准

- 所有交互 UI 都有 normal、loading、success、error 或 disabled 状态。
- Bubble 不遮挡核心操作。
- 食物面板和聊天面板在窄窗口下不溢出。
- 减少动画模式下不出现粒子和大幅位移动画。
- 离线状态下禁用不可用操作并给出原因。

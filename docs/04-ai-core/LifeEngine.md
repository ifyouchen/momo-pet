# Life Engine

## 目标

Life Engine 负责让宠物状态随时间和用户互动变化。

## 状态字段

- hunger：饱食度
- mood：心情
- cleanliness：清洁度
- energy：能量
- intimacy：亲密度
- experience：经验
- level：等级

## 时间衰减

规则：

- 饱食度随时间下降。
- 清洁度随随机事件下降。
- 能量在活动后下降，在睡眠后恢复。
- 心情受饱食度、清洁度、互动影响。

## 行为决策

本地优先使用状态机，不为普通行为频繁调用 LLM。

只有以下场景需要 AI：

- 主动陪伴文案
- 长期记忆召回
- 特殊纪念日
